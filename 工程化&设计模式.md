webpack 构建过程  
 读取配置 → 编译模块 → 生成依赖图 → 打包输出

loader 和 plugin 区别  
Loader = 文件转换器，把非 JS 文件（CSS/图片等）转成 JS 能处理的模块  
Plugin = Webpack 的功能扩展器，在构建过程的不同阶段插入自定义逻辑（生成 html 文件，压缩代码，打包分析等）
