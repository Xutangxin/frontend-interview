em，rem，px 区别  
px 是绝对单位，像素
em 是相对单位，相对于父元素字体大小来算
rem 是相对单位，相对于 html 字体大小来计算

css 定位  
static: 默认值，元素按照正常文档流排列  
relative: 相对定位，相对于原位置进行偏移  
absolute: 绝对定位，相对于最近的非 static 定位祖先元素  
fixed: 固定定位，相对于浏览器窗口  
sticky: 粘性定位，根据滚动位置决定是相对定位还是固定定位

对 BFC 的理解  
BFC 是页面上的一个隔离的独立容器，容器内部的元素不会在布局上影响到外部元素。常见的创建方式有设置 overflow 不为 visible、float 不为 none 等。主要应用在清除浮动、避免 margin 重叠和实现自适应布局等场景。

重绘重排  
重绘：  
当渲染树中的一些元素需要更新属性，而这些属性只是影响元素的外观，风格，而不会影响布局的操作，比如 background-color 就只会触发重绘，而不会触发重排。
重排：  
当渲染树中的一部分(或全部)因为元素的规模尺寸，布局，隐藏等改变而需要重新构建。
优化：
合并多次操作到一次操作中
